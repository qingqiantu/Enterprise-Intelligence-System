# 🏢 企业智能管理系统（Enterprise Intelligence System）  
*一个集成 ERP、知识库与智能分析模块的企业管理平台*

---

## 📘 一、项目简介
**Enterprise Intelligence System** 是一个面向中小企业的信息化管理平台，集成了以下三大模块：
- **ERP 系统**：用于采购、库存与审批流程管理；
- **知识库模块**：支持文档上传、分类、版本控制与智能检索；
- **智能分析模块**：提供 OCR 发票识别、RAG 智能问答与预测性分析。

🎯 **项目目标**：  
通过统一的企业数据平台，实现办公自动化、知识共享与智能化决策。
